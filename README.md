# illusion
Скрипт на Пайтоне, с помощью которого можно создавать невидимые видео.

generator.py - создает анимацию со скоростью 0.1 секунды, fastgenerator.py - создает анимацию со скоростью 0.001 секунды.

Но вы всегда можете внести изменение в строку:

self.switch_interval = 0.001

0.1 сек = 10 смен/сек - видимое мерцание

0.01 сек = 100 смен/сек - быстрое мерцание

0.001 сек = 1000 смен/сек - максимально быстро

Для работы скрипта нужен файл с начальной текстурой (100x100, jpg) и файл с маской (png). Вы можете разместить их где угодно, но проще всего в папке со скриптом, тогда можно будет просто прописывать коротко, без длинных путей, например "01.jpg" и "maska.png".

При создании маски используйте разрешение изображения 1920x1080. Всё, что на нем будет черного цвета - и станет тем самым "невидимым" объектом.

Для запуска скрипта понадобится установка двух библиотек:

pip install Pillow pygame
